Proyecto

NovaTask

Plataforma web para organizacion academica universitaria orientada a estudiantes PUCE. Este tablero resume el diseno completo de experiencia de usuario para el MVP de tesis.

Objetivo UX

Reducir olvidos, bajar la carga mental y mejorar el cumplimiento de tareas con una interfaz visual, cercana y facil de usar en contexto academico.

Secciones

Vision Design System Prototype Lab Figma Canvas Handoff

Entregable

Board UI/UX completo con pantallas de login, dashboard, Kanban, calendario, analiticas, plantillas y version movil.

01 Vision de producto

La plataforma debe sentirse como un centro de control academico, no como una app corporativa.

Tema de tesis alineado

Propuesta de valor

Un solo lugar para ver materias, entregas, proyectos y progreso semanal.

NovaTask se disena para estudiantes que viven entre clases, Moodle, proyectos grupales y fechas de entrega. La interfaz prioriza claridad, urgencia visual y control rapido.

Persona principal

Estudiante PUCE con alta carga

Usa varias materias a la vez, trabaja en grupo, olvida deadlines cuando cambia de contexto y necesita una herramienta rapida, visual y en espanol.

Pain points

  • No ve toda la semana de un vistazo.
  • Olvida fechas entre diferentes plataformas.
  • No sabe que tarea desbloquea otra.
  • Se estresa por acumulacion de pendientes.

UX success

  • Menos tiempo para entender que hacer hoy.
  • Mayor porcentaje de tareas a tiempo.
  • Mejor sensacion de control semanal.
  • Interfaz simple segun SUS.
